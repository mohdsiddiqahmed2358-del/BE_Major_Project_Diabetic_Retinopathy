About Detection Settings

These settings control the sensitivity and parameters of the microaneurysm detection algorithm.

  • Confidence Threshold: Higher values make the detection more conservative (fewer false positives, but may miss some microaneurysms).
  • Minimum Lesion Size: The smallest size (in pixels) that will be considered a microaneurysm.
  • Maximum Lesion Size: The largest size (in pixels) that will be considered a microaneurysm.
